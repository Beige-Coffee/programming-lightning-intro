# Aliases

The workbook comes with a couple of aliases that will be useful as we begin to program Lightning. Each alias executes a Bitcoin remote procedure call (RPC) command under the hood, interacting with the node running in the background of this workbook. These aliases are designed to simplify executing those commands, and links to the underlying documentation for each command are also provided.
